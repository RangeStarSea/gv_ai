## 🧱 一、基础结构标签（HTML 文档骨架）
| 标签              | 说明                            |
| ----------------- | --------------------------------|
| `<!DOCTYPE html>` | 声明文档类型（HTML5）            |
| `<html>`          | HTML 文档的根元素                |
| `<head>`          | 定义文档头部（元信息）            |
| `<meta>`          | 定义元数据（编码、视口、描述等）   |
| `<title>`         | 定义网页标题                     |
| `<link>`          | 链接外部资源（CSS、图标等）       |
| `<style>`         | 定义内部样式表                   |
| `<script>`        | 插入或引用 JavaScript 脚本       |
| `<body>`          | 定义文档主体内容                 |
---
## 🧭 二、结构与语义化标签（HTML5 新语义）
| 标签          | 说明                               |
| ------------- | ---------------------------------- |
| `<header>`    | 页头或区块头部                     |
| `<footer>`    | 页脚或区块尾部                     |
| `<main>`      | 主体内容区域（一个页面只能有一个） |
| `<nav>`       | 导航栏区域                         |
| `<section>`   | 文档的独立章节                     |
| `<article>`   | 独立的文章、博文、帖子等           |
| `<aside>`     | 侧栏、附注信息                     |
| `<address>`   | 联系信息（地址）                   |
| `<h1>`–`<h6>` | 标题级别 1~6                       |
| `<hr>`        | 水平分隔线                         |
---

## 🚧 十三、过时（不推荐使用）的标签
| 标签                                    | 替代方案                     |
| --------------------------------------- | ---------------------------- |
| `<center>`                              | 用 CSS `text-align: center;` |
| `<font>`                                | 用 CSS 控制字体样式          |
| `<big>` / `<small>`                     | 用 CSS 调整字体大小          |
| `<marquee>`                             | 用 CSS 动画                  |
| `<bgsound>`                             | 用 `<audio>`                 |
| `<frame>` / `<frameset>` / `<noframes>` | 已废弃，用 `<iframe>`        |
